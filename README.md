# AI应用开发转型学习

> **适用对象**：Java/Spring Boot企业级开发经验  
> **目标**：完成从"Java开发"到"AI应用开发"的核心技能储备，产出可写入简历的项目作品  
> **日均学习时间**：工作日 2-3小时（晚上）+ 周末 6-8小时  
> **总学时**：约 120-140 小时  

---

## 学习原则

1. **不做平行替代**——你有Java底子，Python只学差异点，不从头学编程
2. **每个模块必须有可运行产出**——不写笔记，写代码
3. **以终为始**——第6周的简历和面试准备倒推前5周学什么
